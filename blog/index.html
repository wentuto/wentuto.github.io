用 GITHUB PAGES + HUGO 打造我自己的BLOG

October 6, 2023 in BLOG
前言 之前已經使用過 GHOST 打造我自己的BLOG，它是一套非常好用的 open source blog，功能也相當強大，有很好用的編輯介面與擴充功能，但也因功能強大，遇到的問題要克服也比較複雜，我在更新 HTTPS 的 SSL 時一直卡關後，原先的 BLOG 就沒有在更新了，再來也是因為要自己 host 網站，會有一筆 host 的費用，當看到可以用 GITHUB PAGES 的方案就想要換這個免費的方式。 評估 後來決定使用 HUGO＋GITHUB PAGES，主要就是『快』跟『簡單』：因為 HUGO 是目前調查的靜態網頁系統中生成速度最快的，可以節省不少時間，所以『快』，再來就是因為相依的模組較少，可避免安裝或升級時引起的問題，所以『簡單』。至於為何不使用以前使用過的 GHOST，主要是雖然 GHOST 功能較強大，但管理邏輯上比較複雜，比較不知從何改起，怕升級失敗後，之前的心血就白費了。最後就是可以省下一筆自己 HOST 網站的費用，想說等之後真的越寫越多，再搬到GHOST也不是不可以 XD。 安裝 首先是在電腦上安裝 HUGO 這個用 Go 編寫的靜態網站生成器，由於我用的電腦是 MACBOOK，所以使用下面 command brew install hugo 這樣就可以安裝完成了。 使用 HUGO 建立新 BLOG
